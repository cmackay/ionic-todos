# Ionic Todos

  Example project structure using:

  - Ionic Framework
  - Angular
  - Gulp
  - Sass
  - Browserify
  - Napa

### Commands

Install Global Dependencies:

    $ npm install -g gulp

Install Project Dependencies:

    $ npm install

Run Development Mode:

    $ gulp

Run Tests:

    $ gulp test

Prepare Build:

    $ gulp build

Clean Build Artifacts:

    $ gulp clean

Clean Build Artifacts (including node_modules):

    $ gulp clean-all
